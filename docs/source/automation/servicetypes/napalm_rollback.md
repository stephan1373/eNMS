Use Napalm to rollback a configuration.

![Napalm Rollback Service](../../_static/automation/service_types/napalm_rollback.png)

Configuration parameters for creating this service instance:

- All [Napalm Service Common Parameters](napalm_common.md). 

There are no specific parameters for rollback. This service allows for rolling back an 
already committed config change that was made with the Napalm Configuration Service.
